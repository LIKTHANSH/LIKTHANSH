## 🚀 Featured Projects

### 🧠 NeuroVision – Eye-Gaze Controlled Smart Home
- Hands-free IoT appliance control using **eye blinks** detected via webcam
- **MediaPipe FaceMesh** tracks 468 facial landmarks; EAR-based blink detection
  triggers serial commands (`RED_TOGGLE`, `YELLOW_TOGGLE` etc.) to **Arduino UNO**
- Color menu scrolls automatically — blink to select and toggle the highlighted appliance
- Built with Python, OpenCV, PySerial — runs fully offline, no cloud dependency
🔗 Repo: https://github.com/LIKTHANSH/NeuroVision

### 👁️ NeuroVision – Eye-Gaze Virtual Keyboard
- Hands-free typing system using **gaze direction** (left/right) and **blink detection**
- **MediaPipe FaceMesh** iris landmarks detect gaze; blink held for 6 frames selects a letter
- Split keyboard layout (QWERTY-based); letters scroll and highlight automatically
- Upgraded from Dlib to **MediaPipe** for zero-dependency landmark detection
- Built with Python, OpenCV, MediaPipe — runs in real-time on any webcam
🔗 Repo: https://github.com/LIKTHANSH/NeuroVision_KeyBoard
